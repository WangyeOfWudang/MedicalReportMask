# 医疗报告脱敏 MedicalReportMask
对医疗报告图片进行脱敏，删除图片上的隐私信息。
上传一张医疗报告图片，自动识别其中的隐私字段，并对隐私字段进行打码。返回一张打码脱敏后的图片。
## 功能列表
- 医疗报告图片脱敏
